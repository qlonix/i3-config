i3wm Ultimate Cheat Sheet

直感的でストレスフリーな操作ガイド
💡 基本ルール: この設定では Mod キーは Windowsキー (Super) に設定されています。

🚀 Basics (基本操作)

ターミナルを起動
Mod + Enter
ランチャー (Rofi) 起動
Mod + d
ウィンドウを閉じる
Mod + Shift + q
i3の再読み込み
Mod + Shift + c
i3の再起動
Mod + Shift + r
設定アップデート (GitHub)
Mod + Shift + u
キーボードレイアウト設定
Mod + Ctrl + k
i3の終了メニュー
Mod + Shift + e

🪟 Focus & Move (フォーカスと移動)

左へフォーカス
Mod + h または ←
下へフォーカス
Mod + j または ↓
上へフォーカス
Mod + k または ↑
右へフォーカス
Mod + l または →
ウィンドウを左へ移動
Mod + Shift + h (or ←)
ウィンドウを下へ移動
Mod + Shift + j (or ↓)
ウィンドウを上へ移動
Mod + Shift + k (or ↑)
ウィンドウを右へ移動
Mod + Shift + l (or →)

📐 Layouts (レイアウト操作)

次を「水平」に分割
Mod + b
次を「垂直」に分割
Mod + v
フルスクリーン切替
Mod + f
スタック(縦並び)にする
Mod + s
タブ(横並び)にする
Mod + w
分割モードの切替
Mod + e
浮きウィンドウ(フロート)切替
Mod + Shift + Space
フロート/タイル間のフォーカス移動
Mod + Space

🖥️ Workspaces (ワークスペース)

WS 1〜0 へ移動
Mod + 1 - 0
ウィンドウを WS 1〜0 へ移動
Mod + Shift + 1 - 0

🔊 System (メディアキー)

音量を上げる
Volume Up
音量を下げる
Volume Down
ミュート切替
Mute
画面を明るく
Brightness Up
画面を暗く
Brightness Down
